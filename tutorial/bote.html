Bote

Para controlar los gastos podéis utilizar un bote, o no...Pero os ofrecemos las siguientes posibilidades:

Físico

Si elegís esta opción, significa que existe un bote real en el que insertaris dinero.

Por tanto, sólo se podran realizar compras si hay dinero dentro de él.

Débito

Esta opción no necesita de bote físico.

Si elegís esta opción, cada vez que un compañero haga la compra de objetos comunes, FlatMate se encargará de calcular el dinero que le debe el resto.

No necesitamos bote

Significa que sois responsables
